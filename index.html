Ramses

the API generation framework

✓  Automatically generate APIs from RAML files
✓  With API docs
✓  And JavaScript clients
✓  Powerful queries in the URL with Elasticsearch
✓  Advanced auth and fine-grained ACLs
Step 1, describe an API:

/monuments:
  get:
  post:
    body:
      application/json:
        schema: |
          { "$schema": "http://json-schema.org/schema",
            "type": "object",
            "properties": {
              "name": { "type": "string" },
              "location":  { "type": "string" },
              "dynasty": { "type": "integer" }
            }
          }
Step 2, use it:

$ curl http://localhost/api/monuments
{
  "name": "Abu Simbel",
  "location": "Nubia",
  "dynasty": 19
}
