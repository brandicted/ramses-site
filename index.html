Ramses

APIs for immortals

Step 1, describe an API:

/monuments:
  get:
  post:
    body:
      application/json:
        schema: |
          { "$schema": "http://json-schema.org/schema",
            "type": "object",
            "properties": {
              "name": { "type": "string" },
              "location":  { "type": "string" },
              "dynasty": { "type": "integer" }
            }
          }
Step 2, use it:

$ curl http://localhost/api/monuments
{[{
  "name": "Abu Simbel"
  "location": "Nubia",
  "dynasty": 19
}]}
Get started

Rapid API creation

One text file to rule them all

Using RAML, Ramses generates production-ready APIs. No code required. Generate frontend code, documentation and tests.

Elasticsearch-powered

Advanced querying

Elasticsearch query DSL for your endpoints. Aggregations, full-text search, ranges, sorting, limits, pagination and more.

Fine-grained security

Single Source of Truth & full access control

Syncs with Postgres or Mongo to ensure data safety. Mutiple authentication schemes. Collection and object-level authorization.
